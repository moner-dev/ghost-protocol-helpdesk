# Changelog

All notable changes to Ghost Protocol will be documented in this file.

## [1.0.0] - 2026-04-09

### Added
- Full incident management system with CRUD operations
- RBAC system with 4 roles: OWNER, ADMIN, OPERATOR, VIEWER
- Knowledge Base with article management and feedback system
- End Users Management with ticket reassignment on delete
- Company Departments with incident reassignment protection
- User Administration with role hierarchy enforcement
- Audit Log with full operation tracking
- Reports and Analytics with PDF/Excel export
- Backup and Restore system (OWNER only)
- KB Notifications bell with real-time updates
- Real-time Dashboard with Department Load chart
- KB feedback system (YES/NO rating + Report an Issue)
- Auto-logout on app close via sessionStorage
- Electron desktop app with Inno Setup installer
- Cinematic splash screen with skip functionality
- Dark theme UI with Ghost Protocol branding
